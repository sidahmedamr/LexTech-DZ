# LexTech-DZ
🚀 LexTech DZ is a complete LegalTech platform for my Master's graduation project. It bridges citizens with certified lawyers in Algeria through an intuitive web interface.

✨ Features:
• 👨‍⚖️ Lawyer verification with golden badge
• 💬 Public legal forum with verified answers
• 📅 Premium consultation booking system
• 💳 Edahabia/CIB payment simulation (20% platform / 80% lawyer)
• 🌐 Multi-language support (English, Français, العربية)
• 🌙 Dark/Light mode toggle
• 📱 Fully responsive design

🛠 Tech Stack: HTML5, CSS3, Vanilla JavaScript
